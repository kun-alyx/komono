30日チャレンジ感想　11～20日目 (Pixiv Fanbox)

Published:
2024-01-23 07:46:50
Imported:
2024-05

Content

　30日チャレンジ、なんとか20日を超え残り10日となりました。これ書いてる時点で22日目なので残り約1週間。ちょっとさすがに絵を描くことに飽きてきました。ということで、11日目から20日目までの感想書いていきます。


　11日目　アームド・ドラゴンLv7 / 遊戯王

　初日だけ18時でそれ以降は毎日20時に投稿してるんですが、こいつだけ20時に遅れてしまいました。悔しい。線画に時間かかったのもありますが、塗りにも迷走しちゃってかかった時間の割に塗りの密度があんまり高くできなかったです。ちなみに他のどのキャラより資料が少なかったです。アニメ出てたはずなのにスクショ全く見なかった。


　12日目　主人公 / ドラゴンクエストビルダーズ

　説明しすぎない場面説明やってみたくて描きました。ラダトームの月面みたいなライティングが好きで、灰色の地面に濃い影だけで伝わるんじゃないかな～と挑戦しました。ただ肝心の影に違和感あったのでちょびっとだけ加筆しました。


　13日目　でじこ / デ・ジ・キャラット

　｢デ・ジ・キャラットにょ｣とかいう子供向けアニメの皮を被ったサブカルオタクアニメがあったせいで、ぼくの人生の結構早い段階からオタクの道に舵を切ってました。あの年代の女の子の顔のバランス好きです。長年秋葉原の顔飾ってただけあってほんといいキャラデザですよね。


　14日目　スコット / スコット・ピルグリム テイクス・オフ

　今回描いたキャラの中でダントツで新しい作品のキャラですね。ネットフリックス独占配信中です。世界観と話の方向性を理解するまでがちょっとアレですが、絵柄かわいいしキャラもみんな良いのでおすすめです。

　あのアニメの絵の面白いところで、ピンボケの描写が人間の目の構造のボケ方をしてるんですよね。カメラの普通にボヤっとしたボケ方じゃなくて、像が左右にズレて二重になってるんです。独特な空気感の出し方が好きで、そういうとこ意識してみました。


　15日目　ルカリオ / ポケットモンスター

　ルカリオって120cmしかないんですよね。映画とかでの使い手の印象もあって大人と組まされがちな気もするんですが、ちょうど7,8歳くらいの子供と目線が同じなんですよね。加えて、波導で共感能力もあるので子供と一緒に本も読めるんじゃね？と思ってこうなりました。ちなみに部屋の雰囲気はリオルのとあるポケカの絵柄を意識しました。


　16日目　インクリング / スプラトゥーン

　3でのマイイカくんです。トラストも持ち武器。ほんとはヤコバイザー被ってるんですが、たなびくゲソと頭のキラキラ描きたかったのでオミットしました。これラフ自体は30チャレ始めるずっと前に描こうと思って作ってあったのを再利用したんですが、トラストむずすぎて作画時間はあんまり短縮されませんでした。幸い、トラスト水鉄砲買ってたので最終手段で写真撮って欲しい形に歪めて描けました。


　17日目　木之本さくら / カードキャプターさくら

　ショタコンなんだから素直に小狼くん描けよというイマジナリーフォロワーの声を聞きながら描きました。でもどうしてもさくらちゃん描きたかったんです。ふかふかのスカートとフリルたいへんだったけど楽しかった。ちなみに表情はミュシャの｢JOY｣の絵をリスペクトしました。

　あと余談ですが、これのあとに小狼くんのらくがき載せたらさくらちゃんの何倍もリアクションつきました。


　18日目　クレイグ・タッカー / サウスパーク

　サウスパークで一番好きなキャラです。顔が一番好み(強めの幻覚を見てる)。ダイナミックなカメラワークにしたくてこの構図にしたんですけど、あんまり見栄えには貢献しなかったかな。これなぜか𝕏からセンシティブ判定食らって警告表示されました。ムカついたので即異議申し立てしたんですが通らず。ファッキューイーロン。


　19日目　カク / ポンコツクエスト

　傲慢で不遜で独善的なキャラが好きなんですが、彼がまさにそんな感じでとてもツボ。第4話だったかの飲み会の絵がなんかちょうどよくて、Chromeの新しいタブページの背景にしてるんです。その場面描いてみました。こういうおとなしい色を上手く使えるようになりたい。


　20日目　脳噛ネウロ / 魔人探偵脳噛ネウロ

　これ描くために資料漁ってたらめちゃくちゃ読み返したくなりました。良い作品ですよね。アニメ化されないかな。

　原作で地面じゃないところに立ってるシーン多いし、そんな感じで1話の初登場シーンみたいなダイナミックな構図にしてもよかったかなと今思いました。けどやっぱ初期の表紙の独特な上下逆さの絵がやっぱり印象的でこうなりました。



　今回もまあまあ長くなりました。お疲れ様です。

　20日終えた時点での感想として、17日目あたりから肩に刺すような痛みが走ってずっと湿布貼ってました。原稿やってたときもガムシャラに絵を描けるときは常に絵を描いてたはずなんですが、今の方が圧倒的に身体とメンタルに来てる気がします。身体の方は落ち着いてきたんですが、メンタルは慣れなそう。0を1にする一番しんどい行程を毎日やるのがきつい。

　とはいえ終りが見えてきたので走り切るのみです。がんばろ～！
