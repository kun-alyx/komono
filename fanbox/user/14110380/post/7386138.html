30日チャレンジ完走した感想　21～30日目 (Pixiv Fanbox)

Published:
2024-01-31 13:11:31
Imported:
2024-05

Content

　おかげさまでなんとか完走できました！！わーい👏ということでいつもの通り個別のイラストの感想と、最後に全体を通しての感想を書いていきます。


　21日目　浦島虎徹 / 刀剣乱舞

　最初はモンハンのバゼルギウス描こうと思ったんですが、ラフ描いた時点でなんか嫌気が差して急遽浦島くんになりました。虎徹の真作をつかまえてやらせることがお芋掘り。お顔がとってもかわいいし髪型がわりとコロコロ変わるのが好きです。髪がどうしても元絵みたいに綺麗に描けなかったんですが、スポイトして比べてみたら元絵の彩度高すぎてびびりました。自分けっこう彩度高い方だと思ってたんですがまだまだでした。


　22日目　ブラック☆スター / ソウルイーター

　キャラデザがすっごいツボです。性格も。声も。そもそもソウルイーターの絵柄が好きなんですが、ブラック☆スターくん格別ですね。これ思ったより伸びたのでみんなも好きだったかぼくの絵柄にハマったんでしょうね。絵としてはフカン苦手なので練習しました。なんか違和感あるからまだ練習必要ですね。


　23日目　アリス / アリスインナイトメア

　知る人ぞ知る名作ゲーム、アリスインナイトメアから主人公アリスです。実はキャロルの原作アリスも大好きなので、ギリギリまで原作描くかこれ描くかで迷いました。テニエル風のタッチもすごくやりたかったんですが、肝心のアリスがあの絵柄でかわいく描くのむずすぎてこっちになりました。この雰囲気の絵も描いてみたかったですしね。


　24日目　スパイダーマン / スパイダーマン

　描かなきゃなるめェよという義務感みたいのがありました。描いてみて改めて実感する洗練されたデザイン。ほんとカッコいい。しかし背景のほうが時間かかりましたねこれ。クリスタに魚眼パース定規あるんですが、使い方わからないので一点透視で描いたのを魚眼歪み加工しただけです。なので画面外に消えた部分けっこうあるんですよね実は。


　25日目　オズワルド / しあわせウサギのオズワルド

　親がディズニー好きじゃない影響でぼくもディズニーぜんぜん見てなくて、ついでにミッキーもそんな好きじゃないんですが、オズワルドは大好きです。生い立ちももあってなおさら。シンプルめに描く予定だったんですが、思ったより背景をしっかりめに描いちゃったのでオズワルドにも陰影を補強しました。上手く行って満足。ちなみに背景はシーのオズワルドとグリーティングできるエリア。今もういないんでしたっけ。会いたかった～


　26日目　キレネンコ / ウサビッチ

　ぼくスニーカーフェチなのバレてると思うんですけど、その発端は彼です。中学生の頃ハマってめっちゃくちゃ描いてました。いまもう失踪してしまったちゃまちゃまさんっていう絵描きさんがいて、ハピツリとウサビッチを独特な絵柄でカッコよく描いてて、すごい憧れてたんですよね。未だにこういうキャラ描くときちょっと意識しちゃいます。


　27日目　空条承太郎 / ジョジョの奇妙な冒険

　ぼくが二次創作するくらいハマった作品で頭身高くてマッチョな作品ってジョジョくらいしかないので、実はマッチョ用の絵にかなり影響を受けてます。一時期模写もしてたので顔つきも多少はそれっぽく描けてると思うんですが、なんかちょっと童顔ですね。あとスタプラ地味に顔が安定してない感じがあってむずかった。


　28日目　おめがレイ&おめがリオ / おめがシスターズ

　ぼくが見てる数少ないVの一組です。伝わる人には伝わる絵にしました。せっかくV描くんだからと思ってハッシュタグつけて投稿したら、ご本人たちにRTしていただけて小躍りしました。女の子描くときは特に髪の描き方塗り方を苦労するんですが、塗はともかく線画が垢抜けないですねえ。


　29日目　リンク / ゼルダの伝説 ムジュラの仮面

　ゼル伝が時オカって世界線が3つに分岐するんですが、ムジュラルート結構報われない感があって好きです。ムジュラ自体の終末感も相まってムジュリンが一番好き。逃れられない死に直面する街を何度も眺めて、救えない世界をいくつも捨てて時を遡り続けたら、かなりメンタルどうにかなりそうですよね。中二だからこういう病んだ感じを求めてしまう。月の顔のブツブツ描くのキモくて楽しかった。


　30日目　ブイゼル / ポケットモンスター

　作品はバラバラにしようと思ってたんですが、どうしても描きたくて描いちゃいました。ブイゼル描くために資料探しするたびに思うんですが、作画によってまあまあブスになるのが好きです。絶妙な顔面をしている。ポケモンってやっぱ公式が最王手感あって、ポケカで良い絵が無限にあるのでどうしても意識しちゃいますよね。これもフローゼルのやつ真似しました。


　全体通しての感想！

　と～～～ても疲れた！11~20日の感想でも書きましたが、原稿やってたときよりも身体に来てます。猫背が過ぎて首肩に刺すような痛みが来たり、腹筋周りも縮こまりすぎて揉んだりするとわりと痛くてウケました。リングフィットもやってある程度身体ほぐしてるはずなんですが、間に合ってないんですかね。

　しかしいろんな方に褒めていただけたのがとても嬉しかった！あんまり自分の素の絵柄に愛着がないのが幸いして、絵ごとにコロコロ作風変えてたら｢なんでも描けてすごい！｣みたいなこといっぱい言っていただけました。あと単純に｢絵が上手い｣と言っていただけたのも本当に嬉しかった。満たされる～～！

　当初の目標として、｢いろんなことに挑戦する｣ということを掲げておりまして、

・普段やらない技法を取り入れてみる

・構図にこだわってみる

・色遣いを綺麗にする

というのを意識してたんですが、みなさん見ててどうでしたか…？途中からとにかく描くのに精一杯であんまり頑張れてなかったかもしれない…。けど、背景への苦手意識はちょっと薄れたかもしれないです。ある程度のそれっぽさにする分には意外となんとかなる。色遣いはまじで変わった気しないので、まだまだ練習だあ。



　なが～～～い感想になっちゃいました。ここまで読んでくれてありがとうございます。チャレンジ中、応援してくれた方、感想リプくれた方、リアクションくれた方、そっと見守ってくれてた方、みなさまありがとうございました～～～！いい体験ができました！

　そしてFANBOX/fantia支援してくれてる皆様、今月はちゃんと更新できなくてすみませんでした！！！！2月からは通常通り更新できると思いますので、今後ともよろしくお願いします…！
